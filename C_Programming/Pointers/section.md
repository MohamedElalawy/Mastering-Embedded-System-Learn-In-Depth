| Expression | Action                  | Effect                                                                 |
|------------|-------------------------|------------------------------------------------------------------------|
| `*ptr++`   | **Post-increment pointer** | Returns the current value, then moves `ptr` to the next address.       |
| `++*ptr`   | **Pre-increment value**    | Increments the value at `ptr` and returns the new value.               |
